
# 📊 Análise de Vendas de Restaurante com Excel, SQL Server e Power BI

## 📌 Objetivo

Desenvolver um dashboard para analisar o desempenho de vendas de um restaurante, identificando indicadores importantes como faturamento, ticket médio, clientes mais valiosos, pratos mais vendidos e consumo de bebidas.

---

## 🛠 Ferramentas Utilizadas

* Excel
* SQL Server
* Power BI

---

## 📂 Fonte dos Dados

Os dados foram coletados a partir dos registros reais de vendas de um restaurante familiar.

Período analisado:

* Janeiro de 2026
* Fevereiro de 2026
* Março de 2026

Informações disponíveis:

* Data da venda
* Cliente
* Prato
* Quantidade
* Bebida
* Quantidade de bebidas
* Valor do pedido

---
## 🔍 Desafios e Tratamento dos Dados
* Os dados originais estavam registrados manualmente em cadernos e precisaram ser digitados e estruturados em planilhas Excel.
* Foi realizada a organização dos dados em formato tabular para facilitar a análise.
* Os registros de pedidos sem bebida foram padronizados como "Nenhuma".
* As datas foram ajustadas para permitir análises mensais no SQL Server e Power BI.
* Foi realizada a revisão dos registros para identificar possíveis inconsistências antes da importação para o banco de dados.
---

## 🗄️ Análise em SQL Server

Foram desenvolvidas consultas SQL para responder perguntas de negócio, como:

### Faturamento Total

* Qual o faturamento total do período analisado?

### Ticket Médio

* Quanto cada cliente gastou em média por pedido?

### Produtos

* Quais são os pratos mais vendidos?
* Quais pratos geram mais faturamento?

### Clientes

* Quem são os clientes que mais gastam?
* Quantos clientes únicos realizaram pedidos?

### Bebidas

* Quais bebidas são mais consumidas?
* Quantos pedidos foram realizados sem bebida?

### Desempenho Mensal

* Qual mês apresentou maior faturamento?
* Qual mês apresentou maior quantidade de pedidos?

---

## 📈 Dashboard Power BI

O dashboard apresenta:

### Indicadores (KPIs)

* Faturamento Total
* Total de Pedidos
* Ticket Médio
* Clientes Únicos

### Visualizações

* Faturamento por Mês
* Quantidade de Pedidos por Mês
* Top Clientes
* Pratos Mais Vendidos
* Consumo de Bebidas

---

## 💡 Principais Insights

* O restaurante faturou mais de R$ 32 mil no período analisado.
* Foram registrados aproximadamente 2 mil pedidos.
* O ticket médio foi de R$ 22,03.
* Foram identificados 170 clientes únicos.
* A Galinha Assada foi o prato mais vendido.
* Mais de 1.200 pedidos foram realizados sem bebida, indicando oportunidade para criação de combos e aumento do ticket médio.
